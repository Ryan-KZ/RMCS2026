# RMCS 2026 Training Modules

Training materials for the Kaizen AI system, featuring three hands-on scenarios designed to demonstrate multi-agent workflows in defense applications.

## Modules

**Module 1: Space Domain Awareness** - "Anomaly Hunter"
Investigate a satellite cyber-event by analyzing telemetry, command logs, and image metadata to determine if an anomaly is a hardware glitch or adversary activity.

**Module 2: Predictive Logistics** - "Arctic Resupply"
Analyze inventory and weather data to identify critical supply shortfalls, then auto-generate DD Form 1348 requisitions for emergency resupply before an Arctic storm.

**Module 3: Intel Fusion** - "RFI Crusher"
Build a RAG pipeline to analyze acquisition documents, extract vendor specifications, draft an RFI, and conduct Red Team analysis to identify bias.
